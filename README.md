# 📩 ZeroWord

A powerful real-time chat application built using **MEAN Stack** (MongoDB, Express.js, Angular, Node.js) and **MERE Stack** (MongoDB, Express.js, React.js, Node.js). This app leverages **Socket.io** for real-time communication and features a sleek UI designed with **DaisyUI**.

## 🚀 Features

- **Real-time messaging** 📲 using **Socket.io**
- **Live message counter** 📊 displayed on the sender's profile image
- **DaisyUI-based UI** 🎨 for a clean and modern design
- **User authentication & profiles** 🔒
- **Group & private chat support** 👥
- **Read receipts and online status** ✅🟢

## 🛠️ Tech Stack

- **Frontend**: React.js + DaisyUI
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **Real-time Communication**: Socket.io
